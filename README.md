# 🎯 Catch Me If You Can

A fun and simple interactive web project that plays with user choices 😏

## 🚀 Live Demo

👉 https://amellahdev-glitch.github.io/catch_me_if_you_can/

---

## 💡 Project Idea

This project is a playful webpage that asks the user a tricky question:

> **"Are you ###?"**

The user is given two buttons:

* ✅ Yes
* ❌ No

But here’s the twist 👇

---

## ⚡ How It Works

* ❌ **"No" Button**
  Every time the user tries to click it, the button moves to a different position on the screen.
  It basically "runs away" from the cursor 😆

* ✅ **"Yes" Button**
  When clicked, an alert appears saying:

  > "I knew you would say that 😏"
